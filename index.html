Manage Leaves

This is a leave managing Python and Flask CRUD(CREATE, READ, UPDATE, DELETE) Web Application.

The entered employee details are stored in a sqlite database.

We can add, edit and delete employee details.



How to use Manage Leaves:

1.Click on Employee Details

2.To add employee, click on Add Employee and fill in the details accordingly.

3.To update employee, click on Edit and change the necessary details and click on save.

4.To delete employee, click on Edit and click on delete.
